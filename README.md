# STRINGS

---

## PRESENTATION

String view and String builder implementation in ISO C.

---

## PREREQUISITES

- C99 compiler.

---

## SUPPORTED PLATFORMS

Basically any platform that can run a C compiler.

---

## HOW TO USE

1- Clone the repo.

```sh
git clone https://github.com/itsjustgalileo/strings/
```

2- Either compile it into a library and link against it or compile the source file as part of your project.

**The include path is a include/strings/ should be added to the project includes**
